HOW DID THIS GET GRAPHED?

The HDTGM info graph site

This site displays various graphs and charts about the movies talked about, and the people who guest-host on, the Earwolf podcast How Did This Get Made?

This site is not associated with, nor endorsed by, the podcast How Did This Get Made?, its creators and hosts, the Earwolf Podcast Network, or the E.E. Scripps Company.

This site uses data from TheMovieDb.org (TMDb). This site uses data retrieved from the TMDb API but is not endorsed or certified by TMDb.

The front-end framework used on this site is Materialize. Charts are generated by Charts.js.

The code for generating this site is available on Github. Suggestions, improvements, bug reports, etc. can be submitted there.
